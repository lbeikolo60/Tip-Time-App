# Tip Time 

**Tip Time** is a simple tip calculator app that calculates the tip automatically as you type.

There is **no calculate button** — the tip updates immediately whenever the amount in the text box changes.

## Features

- Enter a bill amount
- Tip is calculated instantly
- No button clicks required
- Clean and easy-to-use interface
- light/dark mode
- changeable tip amount of presets
- tip addition feature (£1, £5, £10, £25, £50)
- percentage addition (10%, 15% and 20%)

## How It Works

1. Type a bill amount into the text field  
2. The app automatically calculates the tip  
3. The result updates in real time
(optional)
4. change mode to light/dark
